Editing a Python script

to edit a python script. you have to right-click the script and under the "Edit with IDLE" section. select the python version you want to edit the script in.
alternatively in the search field in the taskbar. you can search for "IDLE" and hit enter. this will open Python's built-in IDE. then in the toolbar under "File". press the "Open..." button and find the script that you created last time. once you have found the script. double-click on it to open it.

Printing Text

when you run a script. your running it in a terminal. we can display text in the terminal by using "print()". inside your python script. type "print("Hello World!")" and run the script in the terminal. you will notice that it will print "Hello World!" in the terminal.
