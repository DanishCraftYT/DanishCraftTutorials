Editing a Python script

to edit a python script. you have to right-click the script and press the | "FINISH"...
alternatively in the search field in the taskbar. you can search for "IDLE" and hit enter. this will open Python's built-in IDE. then in the toolbar under "File". press the "Open..." button and find the script that you created last time. once you have found the script. double-click on it to open it.
