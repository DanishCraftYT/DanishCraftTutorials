Creating a Python script

to do so. go to your desktop and right-click. under the "New" section. select "Text Document". now give it a name and change it's extension to ".py".

Running a Python script

to run a python script. go to your Desktop folder in File Explorer. in the folder path at the top of File Explorer. type "cmd" and hit enter.

this will open cmd in the folder that was opened in File Explorer. now type "python 'script_name'" and hit enter. this will run the python script.

you will probably notice that nothing happened and that's because we haven't written any code in our Python script yet.
